    breakpoints

    Grupo de variables para determinar los puntos de ruptura de las media query. Se determinan los valores para sm, md, lg, xl.

    variables

    breakpoints

    $breakpoints: (
"sm": 576,
"md": 768px,
"lg": 992px,
"xl": 1200px
);

    Author

    • luciarl

    colors

    Grupo de variables para determinar un color de fondo o texto. Se determinan los valores primary, primary-trans, secondary, tertiary, tertiary-trans, quaternary. contrast, deep-contrast y btn-shadow. Los colores se proporcionaran en su variante hexadecimal. Los colores que llevan el apellido -trans son versiones con transparencia de sus predecesores. Los colores con el prefijo deep- son la version oscura de su predecesor.

    variables

    colors

    $colors: (
    primary:#835249, 
    primary-trans: #835249e9, 
    secondary: #403232, 
    tertiary: #D9B7B0, 
    tertiary-trans: #d9b7b09e,
    quaternary: #F2F2F2,
    contrast: #5B74A6,
    deep-contrast: #21304c,
    btn-shadow: #00000033
);

    Used by

    • [function] color

    Author

    • luciarl

    font-sizes

    Grupo de variables para determinar el tamaño del texto. Se determinan los valores para xxl, xl, lg, mg, sm, xs, xxs y min.

    variables

    font-sizes

    $font-sizes: (
    xxl:6rem,
    xl:3rem,
    lg:2.5rem,
    md:2rem,
    sm:1.8rem,
    xs:1.6rem,
    xxs:1rem,
    min: 0.7rem
);

    Used by

    • [function] size

    Author

    • luciarl
